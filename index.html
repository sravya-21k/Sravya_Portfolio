Planetprosjekt-Merkur

En interktiv nettside om planeten Merkur

Dette prosjektet er en visuelt engasjerende nettside dedikert til planeten Merkur, bygget med HTML og CSS. Den gir informativt innhold om Merkurs egenskaper, atmosfære og unike trekk, supplert med et rent og responsivt design. Prosjektet viser min evne til å lage strukturerte nettsider med godt organisert innhold, stil og layout.

🛠 Verktøy & Teknologier:

  • HTML – Struktur og innholdsoppbygging
  • CSS – Styling og layout

Oppgaveplanlegger - To-Do Liste Applikasjon

En applikasjon for å planlegge oppgaver og motta varsler

Dette prosjektet er en interaktiv to-do liste applikasjon hvor brukere kan planlegge oppgaver med spesifikke datoer og tider, sette påminnelser og motta varsler. Applikasjonen lar brukere legge til, redigere og slette oppgaver, samtidig som de lagres i lokal lagring for vedvarende tilgang. Designet er responsivt og brukervennlig, og prosjektet viser min evne til å implementere funksjoner som lagring og varsler i nettapplikasjoner.

🛠 Verktøy & Teknologier:

  • HTML – Struktur og innholdsoppbygging
  • CSS – Styling og layout
  • JavaScript – Funksjonalitet og interaktivitet
  • Local Storage – Lagring av oppgaver
  • Notification API – Varsler for påminnelser

Ønskelister 🎅🎄

En interaktiv ønskeliste-applikasjon med juletema

Dette prosjektet er en React-basert webapplikasjon hvor barn kan lage og dele sine ønskelister med julenissen. Siden er designet med et festlig juletema, og brukere kan se hvem som har vært snille eller rampete. Ønskelistene håndteres med eksterne datafiler, og prosjektet inkluderer animasjoner og effekter for å skape en magisk julefølelse. Applikasjonen er fullt responsiv og optimalisert for både mobil og desktop.

🛠 Verktøy & Teknologier:

  • React.js – Bygge gjenbrukbare komponenter
  • Styled Components – CSS-in-JS styling
  • JavaScript (ES6+) – Dynamisk funksjonalitet
  • NPM Snowflakes – Animasjoner med snøeffekt
  • GitHub Pages – Hosting av prosjektet
